#Warehouse AI

A Python-based warehouse inventory management system designed to manage inventory, monitor stock levels, generate reports, and store warehouse data using JSON.


#Features

- Check product stock by shelf
- Add inventory
- Remove inventory
- Add new products
- Edit existing products
- Delete products
- Search products
- Display all warehouse shelves
- Low stock detection
- Restock recommendations
- Warehouse statistics
- Warehouse value calculation
- JSON data persistence
- CSV export
- Transaction logging

#Installation

Clone the repository

```bash
git clone https://github.com/onalpsanci/warehouse-ai.git
```

Move into the project

```bash
cd warehouse-ai
```

Run

```bash
python warehouse.py
```
